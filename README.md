
# ETH-JSON-RPC-TESTS

A simple suite of tests which can be used to test an Ethereum JSON-RPC api meets the specification.

Built with Mocha, Supertest, ethers.js and Yarn.


### 1. Install Dependencies
```
yarn
```

### 2. Configure Environment
```
cp .env.example .env
```

### 3. Run tests
```
make test
````
